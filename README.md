 - Proyecto: Resolución de Ejercicios SQL (Sakila)
Este repositorio contiene la resolución de diversos ejercicios técnicos aplicados sobre la base de datos Sakila, enfocados en el análisis de datos y la gestión de un sistema de alquiler de películas.

 - Herramientas y Tecnologías
Motor de Base de Datos: MySQL 8.0

Entorno de Desarrollo: MySQL Workbench

Lenguaje: SQL

 - Conceptos Aplicados
A lo largo de los ejercicios se han implementado las siguientes capacidades de SQL:

Consultas Básicas: Selección de datos, alias y ordenamiento.

Consultas Avanzadas: Uso de lógica condicional.

JOINS: Relaciones complejas entre múltiples tablas.

Operadores Especiales: Filtrado avanzado con BETWEEN, IN y Expresiones Regulares (REGEXP) para búsqueda de patrones.

Funciones de Agregación: Resúmenes de datos con COUNT, SUM, AVG, junto con GROUP BY y filtros de grupo HAVING.

Subconsultas: Consultas para reportes específicos.
